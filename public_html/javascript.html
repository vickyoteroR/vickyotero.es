Objeto Array

  • concat() Concatena (suma) a un Array otros Arrays, creando uno único
  • forEach() Itera sobre el Array accediendo a cada posición
  • Array.of() Declara un Array con los valores pasados como argumentos
  • Array.from() Devuelve cada letra independiente de cada elemento de un array
  • indexOf) Devuelve el índice(posición) del elemento que le indiquemos dentro de un Array
  • join() Convierte un Array en un string, usando como separador el string deseado (/)
  • length Reporta cuantos elementos componen un array
  • slice() Elimina un rango de elementos de un array
  • splice() Añade o elimina los elementos que se deseen a un Array, a partir de la posición deseada
  • shift() Elimina el primer elemento de un array
  • push() Agrega un elemento al final del array
  • pop() Elimina el úlitmo elemento de un array
  • unshift() Agrega un elemento al principio del array
  • reverse() Invierte el orden de los elementos de un array
  • sort() Ordena los elementos de un Array por orden alfabético

  • endsWith() Arroja true or false, a la pregunta de si un objeto acaba con un string concreto
  • startsWith() Arroja true or false, a la pregunta de si un objeto empieza con un string concreto
  • includes() Arroja true or false, a la pregunta de si un string contiene con un string concreto
  • indexOf() Arroja la posición en un string del caracter concreto que le indiquemos
  • substr() Arroja el texto entre dos índices que le indiquemos
  • length Arroja la longitud de un objeto.
  • match() Localiza un string de forma case-intesive
  • replace() Localiza un string concreto y lo sustituye por lo que le digamos
  • search() Arroja la posición de un sring concreto
  • slice() Corta los caracteres que se encuentren en la posición que le indiquemos
  • split() Divide un objeto en el punto que le indiquemos
  • toLowerCase() toUpperCase() Cambia a mayúscula o a minúscula
  • trim() Elimina espacios antes y después en un texto
